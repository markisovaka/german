Читать подробнее

Помощник Сеня - это сервис который может дать точный прогноз погоды, он ещё может сказать время и ответить на бытовые вопросы. Этот голосовой помощник умеет делать очень полезные вещи, например: включать ютуб, открывать сайт кинопоиск. Он так же может: Включать/выключать и перезагружать комп.

Он настолько умён что он может отвечать на некоторые вопросы по типу: текущее время. Он ещё может добавить что то в список задач.

Кто я: меня зовут Герман, я люблю собак.

О проекте: я написал эту программу используя язык программирования PYTHON, используя приложения: FIGMA и VISUAL STUDIO CODE.

Как это работает? я пишу какой то текст в Visual studio code и потом этот текст отображается в том файле(на сайте). я могу импортировать картинку которую я загрузил и она появится в том файле. я могу добавить практически всё в тот файл.

Мои планы: научится делать сайты с использыванием Visual Studio Code. научится создавать голосового помощника с использыванием Py charm.
